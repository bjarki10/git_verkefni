Git spurningar og svör
1. Hvað gera eftirfarandi Linux skipanir?
	cd
	it's used to change the current directory
	cd ~
	it allows you to return directly to the home directory
	ls
	it lists directory contents
	pwd
        it prints the name of the working directory.
	mkdir
	it's used to create directories on a file system.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
	it clones a repository into a new directory
	git log
	it shows commit logs
	git status
	it shows the working tree status
	git diff
	it show changes between commits, commit and working tree
	git checkout
	it switchs branches or restore working tree files
	
 3. Hver er munurinn á eftirfarandi git skipunum:

4. 	Hvað er útgáfustýring (e.version control)? 
	Version control is a system that records changes to a file or set of files over time so that you can recall specific versions later.

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	Working directory is where git is tracking the difference between your working directory and local repository
	Staging area is where you are temporarily stashing your changes before you commit
	Repository is the area you save all your code
